# Changelog

All notable changes to Costguard are documented here. The project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] - Unreleased

- Stabilize the CLI, configuration, JSON schema, rule IDs, suppressions, and exit codes.
- Harden the public GitHub Action and release binary installation path.
- Eliminate dbt config and Jinja span-mapping panic paths found by the pinned Spellbook benchmark.
- Establish production support for Generic SQL, Snowflake, BigQuery, and Trino.
- Add deterministic, signed, locally verified release artifacts.

## [0.1.0] - 2026-05-23

- Initial experimental release.
